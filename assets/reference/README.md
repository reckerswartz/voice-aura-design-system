# Reference / Inspiration Assets

Design reference images from the original Behance project that inspired this design system.

## Source

**Project:** [AI Voice Generator & Text to Speech Website Design](https://www.behance.net/gallery/245235243/AI-Voice-Generator-Text-to-Speech-Website-Design)
**Authors:** Rah Man, UI Farid Hossain, UXify Studio, Murad Hossain, Design Dynamix
**Platform:** Behance
**Tool:** Adobe XD

## Files

| File | Description |
|------|-------------|
| `behance-voice-aura-cover.jpg` | Project cover image (hero section in laptop mockup) |
| `behance-voice-aura-hero-overview.jpg` | High-resolution hero section overview (3840px wide) |
| `behance-voice-aura-full-page-design.jpg` | Complete page design composite — all sections, typography, components, and mobile views |

## Naming Convention

All reference files follow this pattern:

```
behance-voice-aura-{description}.{ext}
```

- **`behance`** — Source platform
- **`voice-aura`** — Project identifier
- **`{description}`** — What the image shows (e.g. `hero-overview`, `full-page-design`, `cover`)

## Usage

These images are **design reference / inspiration only**. They are not used directly in the
design system output. Use them to:

- Compare implementation accuracy against the original design
- Guide future component improvements
- Onboard new contributors to the project's visual direction
